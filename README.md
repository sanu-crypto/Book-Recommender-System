# 📚 Book Recommender System

A Machine Learning based Book Recommendation Web App built using **Streamlit**, which suggests books based on user selection.

---

## 🚀 Live Demo
🌐 https://your-book-app.streamlit.app

---

## 📌 Features

- 📚 Book Recommendation System
- 📖 Shows description
- ✍️ Author details
- ⭐ Ratings
- 🎲 Surprise Me feature
- 🔥 Trending Books
- 🎨 Beautiful UI
- 🌐 Google Books API integration

---

## 🧠 How It Works

- Uses **TF-IDF Vectorization**
- Applies **Cosine Similarity**
- Recommends similar books

---

## 🛠️ Tech Stack

- Python
- Pandas
- Scikit-learn
- Streamlit
- Google Books API
